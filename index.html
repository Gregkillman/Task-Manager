How to Use This Task Manager

Use the input box below to add, delete, or clear tasks from your list.

  1. Type a task name and click Add Task or press Enter to add it.
  2. Type the name of an existing task and click Delete Task to remove the first match.
  3. Click Clear All to remove every task from the list at once.

Tip: delete matches the full task name and ignores uppercase or lowercase differences.
